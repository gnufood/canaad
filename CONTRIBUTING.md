# Contributing

## Prerequisites

- Rust stable (MSRV 1.70)
- `just` — `cargo install just`
- `wasm-pack` — only needed if touching `canaad-wasm`

## Development

```bash
just ci        # full check — mirrors CI exactly
just lint      # fmt check + clippy
just fmt       # auto-format
just test      # native tests
just miri      # memory safety (requires nightly + miri component)
just test-wasm # WASM node tests (requires wasm-pack)
```

## Lints

This workspace runs clippy at deny all/pedantic/nursery. Run `just lint` before pushing — CI will reject anything it flags.

## pkg/

`canaad_wasm*` files are generated by wasm-pack and not committed. `package.json` and `README.md` are hand-maintained — don't overwrite them. Run `just build-wasm` only if you changed `canaad-wasm`.

## Pull requests

- One logical change per PR
- Conventional commits (`fix:`, `feat:`, `chore:`, `docs:`)
- Open an issue first for anything beyond a small fix

## Bugs

Open a GitHub issue or email `bugs@gnu.foo`.
